# Wurm Online Skill Viewer

Графический инструмент для анализа логов прокачки скиллов в игре **Wurm Online**.

Программа:
- читает лог-файлы Wurm Online
- группирует события по сессиям
- показывает суммарный прирост по скиллам
- строит графики прироста (gain per hour)
- автоматически обновляется при изменении лог-файла

Готовую Windows-версию (.exe) можно скачать в разделе **Releases**.

---

## Использование
1. Запустите `SkillViewer.exe`
2. Нажмите **"Выбрать лог-файл"**
3. Выберите лог Wurm Online
4. Выберите дату и сессию
5. Анализируйте таблицу и графики
